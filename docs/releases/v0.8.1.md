# Ritim 0.8.1

## Türkçe

Bu hotfix, mobil arama sırasında masaüstü oynatıcının ve sıranın kaybolmasına yol açan senkronizasyon sorununu giderir.

### Düzeltilenler

- Telefon uygulamasındaki arama artık şarkı çalarken de çalışır.
- Arama, YouTube Music sayfasını yeniden yüklemek yerine kendi uygulama içi yönlendirmesini kullanır; müzik kesilmez.
- Arama veya sayfa geçişi sırasında PC'deki alt oynatıcı çubuğu kaybolmaz.
- Geçici YouTube Music DOM güncellemeleri artık çalan parçayı ve sıradaki parçaları telefondan silmez.
- Yeni arama sonuçlarına eski aramanın gizli içeriklerinin karışması engellendi.

### İndirmeler

- **Windows:** `Ritim-Setup-0.8.1.exe`
- **Android:** `Ritim-Android-v0.8.1.apk`

## English

This hotfix resolves a synchronization issue that could make the desktop player and queue disappear during a mobile search.

### Fixes

- Mobile search now works while a song is playing.
- Search now uses YouTube Music's in-app navigation instead of reloading the page, so playback continues uninterrupted.
- The desktop bottom player bar no longer disappears during search or page transitions.
- Temporary YouTube Music DOM updates no longer clear the current track or queue on the phone.
- Hidden results from the previous search are no longer mixed into new search results.

### Downloads

- **Windows:** `Ritim-Setup-0.8.1.exe`
- **Android:** `Ritim-Android-v0.8.1.apk`
