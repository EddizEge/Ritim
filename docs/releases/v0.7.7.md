# Ritim 0.7.7

Bu sürüm, uygulama içinden bildirilen mobil oynatma ve gezinme sorunlarını giderir.

This release fixes the mobile playback and navigation issues reported from inside the app.

## İndirmeler / Downloads

- **Windows:** `Ritim-Setup-0.7.7.exe`
- **Android:** `Ritim-Android-v0.7.7.apk`

## Türkçe

- Yeni şarkı seçmek için mevcut parçayı önce durdurma zorunluluğu kaldırıldı.
- Mobil Ana Sayfa, Keşfet ve Kitaplık geçişleri YouTube Music’in gerçek gezinme uçlarına bağlandı.
- PC tam oynatıcı ekranında kalsa bile mobil içerik yükleme ve ana sayfaya dönme akışı düzeltildi.
- Şarkı Sözleri sekmesi gerçek YouTube Music sözlerini telefona aktarıyor.
- Benzer sekmesi YouTube Music’in ilgili parça, albüm, oynatma listesi ve sanatçı önerilerini gösteriyor.
- Hızlı ekran geçişleri sıraya alınarak çakışan sayfa yüklemeleri engellendi.

## English

- Selecting a new song no longer requires pausing the current track first.
- Mobile Home, Explore, and Library now use YouTube Music's real navigation endpoints.
- Mobile content loading and returning home work even when the desktop full-player view is open.
- The Lyrics tab now streams real YouTube Music lyrics to the phone.
- The Related tab now shows YouTube Music song, album, playlist, and artist recommendations.
- Rapid navigation is serialized to prevent overlapping page loads.
